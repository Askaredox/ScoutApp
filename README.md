# 🧩 Backend - Scouteca

Backend serverless desarrollado en **AWS Lambda con Python**, diseñado para manejar autenticación, datos y lógica de negocio de la aplicación web **Scouteca**, una plataforma educativa dirigida a dirigentes de Caminantes de la Asociación de Scouts de Guatemala.

## 🚀 Tecnologías principales

- **Lenguaje:** Python 3.12.9
- **Framework:** AWS Lambda (Serverless microservices)
- **Autenticación:** Amazon Cognito (OAuth2 + Hosted UI)
- **Base de datos:** DynamoDB
- **Infraestructura:** AWS API Gateway + AWS Lambda + IAM Roles
- **Hosting frontend:** AWS Amplify (React + TypeScript)
- **CI/CD:** GitHub Actions
